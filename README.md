# 🤖 AI CS Tutor

A frontend website that helps students learn Computer Science concepts with the help of AI.

## Features

- **Topic cards** — Quick overview of key CS subjects (Data Structures, Algorithms, Programming, etc.)
- **AI Chat interface** — Ask questions and get clear, beginner-friendly explanations
- **Dark mode design** — Clean, modern UI

## Tech Stack

- HTML5
- CSS3 (custom properties, CSS Grid, Flexbox)
- Vanilla JavaScript

## Getting Started

Just open `index.html` in your browser — no build tools or dependencies needed!
